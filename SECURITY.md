# Segurança

## Sobre este repositório

Este é um projeto acadêmico executado **localmente via XAMPP**.  
As credenciais visíveis no código (`root` / sem senha) são padrão do XAMPP
e não representam credenciais de um sistema em produção.

## Relatar vulnerabilidades

Encontrou algo? Abra uma issue ou entre em contato diretamente.  
Não publique detalhes de vulnerabilidades em issues públicas.

## Dados de demonstração

O banco de dados inclui dados fictícios inseridos apenas para fins de teste.  
Não há dados reais de alunos ou usuários de nenhuma instituição.
