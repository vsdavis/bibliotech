# BiblioTech

Sistema de Gerenciamento de Biblioteca Escolar desenvolvido como projeto acadêmico para a disciplina de Aplicações para a Web Multiplataforma.

## Tecnologias utilizadas

- HTML5
- CSS3
- JavaScript
- PHP
- MySQL/MariaDB
- XAMPP

## Funcionalidades

- Login de usuários
- Controle de sessão
- Controle de permissões por usuário
- Cadastro, listagem, edição e inativação de livros
- Cadastro, listagem, edição e inativação de alunos
- Gerenciamento de usuários
- Empréstimos e devoluções de livros
- Relatórios

## Como executar localmente

1. Instale o XAMPP.
2. Copie a pasta `bibliotech` para `C:\xampp\htdocs`.
3. Inicie Apache e MySQL no XAMPP.
4. Importe o arquivo SQL da pasta `banco` no phpMyAdmin.
5. Acesse no navegador:

```text
http://localhost/bibliotech
